= Change the Displayed Section Levels

You can adjust the depth of section levels that are displayed in the Table of Contents (TOC) with the `toclevels` attribute.

== Set `toclevels`

By default, the TOC displays level 1 and level 2 section titles.
You can set a different depth with the `toclevels` attribute.

<1> The `toc` attribute must be set in order to use `toclevels`.
<2> `toclevels` is set and assigned the value `4` in the document header.
The TOC will list the titles of the section 1, 2, 3, and 4 levels when the document is rendered.
