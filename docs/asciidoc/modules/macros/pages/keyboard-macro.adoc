= Keyboard UI Macro

IMPORTANT: You must set the `experimental` attribute to enable the UI macros.

== Keyboard macro syntax

Asciidoctor recognizes a macro for creating keyboard shortcuts using the syntax `+kbd:[key(+key)*]+`.
